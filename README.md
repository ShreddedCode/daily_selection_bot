# DailySelection

### Описание

Данный телеграмм-бот выполняет роль карманного справочника. Через API берёт 
из разных сервисов необходимую информацию и позволяет пользователю узнать 
при помощи доступных команд актуальный курс основных валют и погоду в своём 
родном городе.

### Примечание

Следующие команды могут не подходить для вашей операционной системы. Данные 
инструкции выполнены на Windows OS.

Обратите внимание, что в коде присутствуют переменные, определяющие ваш 
API-токен для взаимодействия с телеграмм-ботом, а так же API-ключ для 
сервиса погоды и пример города для получения необходимой погоды.

### Как установить

- Клонировать репозиторий
```bash
git clone https://github.com/ShreddedCode/daily_selection_bot.git
```
- Перейти в нужный каталог
```bash
cd daily_selection_bot
```
- Установить виртуальное окружение
```
python -m venv venv
```
- Активировать виртуальное окружение
```
source venv/Scripts/activate
```
- Установить все зависимости
```
pip install -r requirements.txt
```
- Запустить бота
```
python daily_selection.py
```
### Технологии
Python 3.9
